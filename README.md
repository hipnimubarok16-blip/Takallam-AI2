# 💬 Takallam AI - Teman Belajar Bicara Anda

Takallam AI adalah aplikasi berbasis web interaktif yang dirancang khusus sebagai tutor virtual bahasa Arab untuk melatih **Maharah Kalam** (keterampilan berbicara) siswa kelas 10 Madrasah Aliyah (MA). Aplikasi ini memanfaatkan kekuatan **Google Gemini API (Gemini 1.5 Flash)** dan dibangun menggunakan **Streamlit**.

---

## 🚀 Fitur Utama
* **Tutor Interaktif**: Memberikan respon bahasa Arab yang fasih dan berharakat.
* **Umpan Balik Langsung**: Memberikan koreksi atau feedback singkat dalam bahasa Indonesia setelah siswa menjawab.
* **Tiga Mode Pembelajaran (Topik MA Kelas 10)**:
  1. *التعارف* (Perkenalan Diri)
  2. *الحياة اليومية في المدرسة* (Kehidupan Sehari-hari di Sekolah)
  3. *النشاطات في العطلة* (Aktivitas di Hari Libur)
* **Manajemen Sesi**: Fitur reset percakapan dan deteksi pergantian topik otomatis.
